# Marketing A/B Test Analysis

## Overview
End-to-end A/B test analysis on a marketing campaign dataset (~588,000 users). This project determines whether showing real ads converts users better than showing a Public Service Announcement (PSA), using statistical hypothesis testing, confidence intervals, and exploratory data analysis.

## Tools Used
- Python (pandas, scipy, matplotlib, seaborn, numpy)
- SQL (SQLite)
- Statistics (Chi-square test, confidence intervals, conversion rate analysis)

## Key Findings

### Statistical Result
- **Ad group conversion rate:** 2.55% vs **PSA group:** 1.79%
- **Absolute lift:** +0.77 percentage points
- **Relative lift:** +43.09% — ads drive 43% more conversions than PSAs
- **Chi-square p-value:** < 0.000001 — result is **statistically significant**
- **95% Confidence Interval:** [0.60%, 0.94%] — entirely above 0, confirming ads work

### Best Times to Show Ads
- **Best days:** Monday (3.32%) and Tuesday (3.04%) have the highest conversion rates
- **Best hours:** 4pm–8pm consistently outperforms other times (3.09% at 4pm)
- **Weekdays outperform weekends** for both groups

### Ad Exposure Effect
- Users who saw **50–99 ads** converted at **11.4%** — significantly higher than low-exposure users
- Users who saw **100–199 ads** converted at **17.7%** — highest conversion segment
- Minimal exposure (0–9 ads) only converts at **0.3%** — frequency matters
